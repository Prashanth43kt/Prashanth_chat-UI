# AI Chat UI Project

Name:- Prashanth K T

>> Description
This project is a responsive AI Chat User Interface built using HTML, CSS, Bootstrap, and JavaScript.

It simulates a chat application with features like:
- Chat history (multiple chats)
- New chat creation
- Welcome screen with suggestion cards
- User and AI message interface
- Typing animation
- Responsive design for desktop, tablet, and mobile

>> Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (jQuery)

>>How to Run
1. Open the project folder
2. Double click on `index.html`
3. OR run using Live Server in VS Code

>> Screenshots
Screenshots for Desktop, Mobile, and Tablet views are available in the `screenshots` folder.

**Note
